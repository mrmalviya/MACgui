## ***General info***
This project is GUI based which allows you to change MAC address. 
Also you can change MAC address periodically .
	
## ***Technologies***
Project is created with:
* Python3
* PyQt5
* macchanger
	
## ***Setup***
To run this project :

```
$ cd MACgui
$ python3 setup.py
$ python3 maccgui.py
```
To run Terminal interface of periodic mac changer :
```
$ python3 macter.py
